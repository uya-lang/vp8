## Actual Error
The generated C uses `sizeof(i16)`, and host `cc` reports `error: 'i16' undeclared`.

## Expected Behavior
`@size_of(i16)` should compile to a valid C type such as `sizeof(int16_t)` or a constant value, and the test should pass.
